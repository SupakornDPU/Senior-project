เรียนรู้ภาษา Python

ฝึกเรียนรู้ไวยากรณ์ของ Python

โดยใช้ Flash Card สำหรับการเรียนรู้
Flash card (แฟลชการ์ด)
คือบัตรหรือแผ่นกระดาษที่ใช้ในการเรียนรู้และจดจำข้อมูลโดยทั่วไปจะมีคำถามหรือข้อมูลด้านหน้าและคำตอบหรือข้อมูลที่เกี่ยวข้องด้านหลังเป็นเครื่องมือที่ช่วยในการฝึกฝนและทบทวนความรู้อย่างมีประสิทธิภาพในหลายๆ รายวิชาและสาขาวิชาต่างๆและมักนำมาใช้ในการเรียนภาษา, คณิตศาสตร์, และหลายสาขาวิชาอื่น ๆ อีกด้วย.
Python (ไพธอน)
คือภาษาโปรแกรมคอมพิวเตอร์ที่ใช้ในการพัฒนาและเขียนโปรแกรมมีความอ่านง่ายและเป็นที่นิยมในการเรียนรู้การเขียนโปรแกรมเนื่องจากไวยากรณ์ที่เข้าใจง่ายและมีชุดคำสั่งที่หลากหลายสำหรับการแก้ปัญหาต่าง ๆ บนคอมพิวเตอร์ได้อย่างมีประสิทธิภาพและรวดเร็ว.
